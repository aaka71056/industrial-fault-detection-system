# Modbus Register Map

## Holding Registers
- 40001 = Temperature Value
- 40002 = Current Value
- 40003 = Vibration RMS
- 40004 = Fault Code
- 40005 = Fault Severity
- 40006 = System Status

## Fault Codes
- 0 = Normal
- 1 = Overtemperature
- 2 = Overcurrent
- 3 = Excessive Vibration
- 4 = Bearing Fault Signature
