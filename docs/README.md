# Project Documentation

## Introduction
This project is designed for industrial equipment fault detection and predictive maintenance.

## Working Principle
The system reads vibration, temperature, and current data from sensors connected to an STM32 microcontroller. The embedded firmware processes the data, checks thresholds, performs FFT analysis for vibration signals, and reports system status over Modbus RTU.

## Applications
- Rotating machinery monitoring
- Predictive maintenance
- Motor fault detection
- Industrial automation systems
