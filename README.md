## Overview
This project implements an industrial equipment fault detection system using STM32F303RE and a Python monitoring interface. The system monitors vibration, temperature, and motor current, processes sensor data, and identifies abnormal machine conditions for predictive maintenance applications.

## Features
- Vibration monitoring using accelerometer
- Temperature monitoring using PT100 interface
- Current sensing for motor load analysis
- Fault detection logic
- FFT-based vibration analysis
- Modbus RTU communication
- Python monitoring dashboard
